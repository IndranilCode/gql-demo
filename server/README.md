# GraphQL Server

Server implementation files for the GraphQL Demo project.

## Files

- `server.js` - Basic server setup
- `server2.js` - Extended server with multiple queries and resolvers
- `package.json` - Dependencies and scripts

## Quick Start

```bash
npm install
npm run dev
```

For complete documentation including GraphQL schema, queries, and examples, see the [main README](../README.md).
