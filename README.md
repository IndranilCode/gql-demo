# GraphQL Demo

A GraphQL server demo project built with Apollo Server and Express.js.

## Project Structure

```
gql-demo/
└── server/          # GraphQL server implementation
    ├── server.js    # Basic server setup
    ├── server2.js   # Extended server with queries
    ├── package.json
    └── README.md    # Server-specific documentation
```

## Getting Started

Navigate to the server directory and follow the instructions:

```bash
cd server
npm install
npm run dev
```

See [server/README.md](server/README.md) for detailed documentation.

## Features

- GraphQL API with Apollo Server
- Express.js integration
- Auto-restart with nodemon
- Sample queries with resolvers

## Tech Stack

- Node.js
- Express.js v4
- Apollo Server v3
- GraphQL
